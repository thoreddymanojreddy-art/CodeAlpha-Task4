📊 Sentiment Analysis on Amazon Product Reviews

CodeAlpha Data Analytics Internship – Task 4

📌 Project Overview

This project performs Sentiment Analysis on Amazon product reviews using two popular Natural Language Processing (NLP) techniques:

VADER (Valence Aware Dictionary and Sentiment Reasoner) – Rule-based sentiment analysis.

TextBlob – NLP-based sentiment analysis using polarity and subjectivity scores.


The project analyzes customer reviews, classifies them into Positive, Negative, and Neutral sentiments, and visualizes the results using multiple charts and a word cloud.


---

🎯 Objectives

Clean and preprocess review text.

Analyze sentiment using VADER and TextBlob.

Compare sentiment classifications from both models.

Generate visual insights from review data.

Save sentiment analysis results for further use.

Install the required libraries:

---

▶️ How to Run

Execute the script:

python task4_sentiment_analysis.py

The program will:

1. Load review data.


2. Clean text data.


3. Perform sentiment analysis using VADER.


4. Perform sentiment analysis using TextBlob.


5. Generate visualizations.


6. Save results to a CSV file.




---

📊 Visualizations Generated

1. Sentiment Distribution

Displays the count of Positive, Neutral, and Negative reviews.

2. VADER Score Histogram

Shows the distribution of sentiment scores calculated by VADER.

3. Polarity vs Subjectivity Scatter Plot

Visualizes TextBlob polarity and subjectivity values.

4. Word Cloud

Highlights the most frequently used words in customer reviews.
